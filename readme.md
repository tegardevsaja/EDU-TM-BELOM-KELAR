*ABSENSI*
Berfungsi untuk absen siswa untuk Program Terpadu (PT), dimana nanti guru terkait program terpadu dapat melakukan absen di halaman ini, dan nantinya jika ada siswa, tidak mengikuti program terpadu dalam kondisi alfa/bolos lebih dari 3 kali maka sistem akan membaca otomatis saat cetak sertif siswa itu tidak dapat mendapatkan sertifikat. kondisi lain siswa seperti :
- Sakit = Max 5 Kali
- Izin = Max 5 kali
- Alfa/Bolos = Max 3 kali 

jika sudah lebih dari batas maxsimal yang sudah di tentukan maka, sistem otomatis membaca siswa tidak akan mendapatkan sertfikat.
